1 #classes Classes

Re-usable utility classes.

1.1 #classes.cols .cols

A columns container. Its child elements are columned inexplicitly.

Example
Left Column
Right Column
Column #1
Column #2
Column #3
Column #4
<!-- 2 columns -->
<div class="cols">
	<div>Left Column</div>
	<div>Right Column</div>
</div>
<!-- Any number of columns are OK -->
<div class="cols">
	<div>Column #1</div>
	<div>Column #2</div>
	<div>Column #3</div>
	<div>Column #4</div>
</div>

1.2 #classes.cols-n .cols-{n}

A container for specific number of columns.

Example
Column #1
Column #2
Column #3
Column #4 (wrapped)
Column #5 (wrapped)
<!-- 3 columns -->
<div class="cols-3">
	<div>Column #1</div>
	<div>Column #2</div>
	<div>Column #3</div>
	<div>Column #4 (wrapped)</div>
	<div>Column #5 (wrapped)</div>
</div>
